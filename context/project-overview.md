# Project Overview: Rahi's Collection

## Description
A blazing-fast, Gen-Z-coded website for **Rahi's Collection** — a family retail shop in Ghungoor, Silchar selling clothes, jewellery, and stationery. The site serves as a digital storefront and SEO funnel to drive walk-in customers and WhatsApp orders.

## Goals
1. **Rank in Top 3** for "clothes shop in Silchar" on Google Maps.
2. **40+ WhatsApp enquiries/month** from the website.
3. **2,000+ unique monthly visitors** (mostly local mobile users).
4. **Lighthouse 95+** on all performance and SEO metrics.
5. **<1.5s LCP** on 4G mobile networks.

## Core User Flow
1. **Discovery**: User finds the shop via Google Search ("saree shop Silchar") or Google Maps.
2. **Engagement**: User browses the mobile-friendly catalog/lookbook to see latest collections.
3. **Conversion**: User clicks the WhatsApp button to enquire about a specific item or price.
4. **Action**: User visits the physical shop in Ghungoor using the embedded map/directions.

## Features
- **Visual Catalog**: Browse categories (Ladies, Gents, Kids, Jewellery, Stationery).
- **Seasonal Lookbooks**: High-quality photo essays for festivals (Durga Puja, etc.).
- **WhatsApp Integration**: Persistent float button and per-product enquiry links.
- **Visit Us**: Interactive map, hours, and directions.
- **Local SEO**: Optimized metadata and local business schema.

## In-Scope
- Next.js 15 App Router implementation.
- Responsive, mobile-first design (Gen-Z boutique aesthetic).
- File-based content (MDX) for collections and lookbooks.
- Basic bilingual touches (English + Bengali).

## Out-of-Scope
- Online checkout/payment gateway (Phase 2).
- User accounts or shopping carts.
- Backend Admin CMS (Phase 2).
- Full multi-language translation.

## Success Criteria
- [ ] All pages score 95+ on Lighthouse.
- [ ] LCP is under 1.5s on mobile throttled to 4G.
- [ ] WhatsApp button is easily accessible on every page.
- [ ] LocalBusiness schema is correctly implemented and verified.
